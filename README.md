# 論文アトリエ

教育思想史研究、教育哲学研究、美術教育学研究向けの論文執筆補助アプリです。

## 開き方

PCだけで使う場合は `index.html` をブラウザで開きます。

スマホからも使う場合は、同じWi-Fi上でローカルサーバーを起動します。

```bash
cd /Users/ko/Documents/Codex/2026-05-13/new-chat
node server.mjs
```

AI相談機能でOpenAI APIを使う場合は、APIキーを設定してから起動します。既定モデルは `gpt-5.5-pro` です。

```bash
cd /Users/ko/Documents/Codex/2026-05-13/new-chat
OPENAI_API_KEY=sk-... node server.mjs
```

表示された `スマホ: http://...:4173` のURLをスマホで開いてください。
Macでは `start-local-server.command` をダブルクリックして起動することもできます。

外出先で使う場合は、クラウドへ配置するか、自宅PCを安全なトンネルで公開します。手順は `DEPLOY.md` を見てください。

## できること

- 論文テーマ候補の整理
- 複数研究プロジェクトの切り替え編集
- 途中経過のスナップショット保存と復元
- JSON読み込みによる別端末への研究データ移行
- テーマ設定から出力までの10工程ワークフロー
- 工程ごとのAIチャット・助言・文章案作成
- 論文の型に基づくアウトライン生成
- 先行研究・一次資料マトリクスの作成
- 公開学術メタデータ検索から論文候補を取り込み
- 検索結果・登録文献から先行研究整理の叩き台とAI相談文を作成
- 先行研究の課題からアウトラインを再検証
- 節ごとの本文編集
- パラグラフライティングに基づく段落機能の自動判定・色分け
- 必要文献の割り当てと引用表記の挿入
- 査読コメントと修正方針の生成
- 要旨の作成
- ファクトチェック・校正メモ
- Markdown、JSON、Word（`.docx`）出力

入力内容は研究プロジェクトごとにブラウザの `localStorage` に自動保存されます。別端末で完全に同じデータを使いたい場合は、JSONを書き出して保管してください。
